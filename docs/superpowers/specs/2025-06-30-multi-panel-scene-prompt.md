# 场景多面板布局参考图提示词

> 日期: 2025-06-30 | 状态: 已确认

## 目标

在场景提取输出中增加第三种提示词类型：多面板布局参考图（GPT-Image-2 专用），与现有全景版、俯视图版并列。

## 改动文件

| 文件 | 改动 |
|------|------|
| `knowledge_base/Scene_Extraction_Skills_v5.1.json` → v5.2 | 新增 `multi_panel_gpt` 模版 |
| `extractors/scenes.py` `DETAIL_SYSTEM` | 输出格式增加 `multi_panel_gpt` 字段 |
| `templates/report.html` | 场景详情增加多面板卡片（含复制/修改按钮） |

## 多面板提示词模版

```
多面板 {场景名称} 概念设计稿，同一场景的多视角呈现，
顶部为 {场景主广角全景}，中间行为 {关键空间分区缩略图}，
底部为 {整场纵览大图}，附带 {2-3个特写细节插图}，
{场景类型标签}，影视美术设计拆解，清晰的分镜排版

场景与物件
{从十层提取：空间骨架 + 材质 + 陈设道具 + 标识}

光影与色调
{从十层提取：光源位置色温 + 实拍氛围 + 光影方案}

画质与风格
超写实，超高细节，精细的环境细节，真实的材质纹理，
8k，高分辨率，电影级构图，Artstation 热门风格，
大师级概念艺术，影视美术设计，物理基于渲染
```

## 约束

- 仅输出 GPT-Image-2 版本（不输出 Nano Banana 2 Pro 版本）
- 场景物件和光影由 AI 根据剧本内容从十层数据中自由组织
- HTML 输出附带复制和修改按钮（与全景/俯视图一致）
- 模版底部固定画质与风格段，不依剧本变化

## 输出顺序

全景版 Nano → 全景版 GPT → 俯视图 Nano → 俯视图 GPT → **多面板布局 GPT**
